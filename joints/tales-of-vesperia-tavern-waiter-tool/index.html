Overview

This is a tool designed to help with rembering an order during the tavern waiter mini-game in Tales of Vesperia. Items are displayed in the same order as in the game, which should increase menuing speed. It has several features designed for easy use.

Features

Quick Selection

Clicking anywhere on an item will select the corresponding quantity field.

Keyboard Selection

Selections can be made without requiring mouse input by typing the two-letter shortcut for any item. For example, typing ra will select the quantity field for Rappig Steak.

Quick Reset

Pressing Tab then Space or Return at any point will quickly reset the entire menu.

Self-Contained

Everything needed to use the tool is contained in this page. It can be downloaded and used offline if desired.
